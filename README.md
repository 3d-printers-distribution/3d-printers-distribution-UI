# 3d-printers-distribution

## Project setup
```
yarn install
```

Add environment variable `VUA_APP_FB_API_KEY` with the Api Key for Firebase.

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
